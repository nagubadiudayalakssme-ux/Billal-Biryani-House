# 🍗 Billal Biryani House

A modern, responsive restaurant menu web application built using HTML and CSS.  
This project presents menu items in a clean, visually appealing layout with enhanced readability, a light-themed design, and QR code access for quick navigation.

---

## 🌐 Live Demo

https://nagubadiudayalakssme-ux.github.io/Billal-Biryani-House/

---

## ✨ Features

- Light themed attractive UI  
- Big and readable fonts  
- Clean and structured layout  
- Mobile responsive design  
- QR code for instant access  
- Smooth hover effects  

---

## 📱 QR Code

Scan the QR code on the website to open the menu instantly.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- QR Code API
